This js utility simply could be use to replace a specific word from names multiple folders. 

Eg. folders names - 1. ayush_www 
                    2. ayush_123 
                    3. ayush_coding   

Replacing the word 'ayush' with 'anexas'

New folders names - 1. anexas_www 
                    2. anexas_123 
                    3. anexas_coding
